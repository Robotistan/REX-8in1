## REX Servo Kalibrasyon Adımları

* 1 - Dosyayı REX ana kartına yükleyin.
* 2 - Servo motorları REX ana kartına bağlayın.
* 3 - Servo kollarını servo motorlara takın.
* 4 - Servo durana kadar servo motorları saat yönünde çevirin.
* 5 - Kablolar 180 dereceyi, tam karşı taraf ise 0 dereceyi gösterir.
* 6 - Servo kollarını yeni pozisyona getirin. Şimdi servo motorlar 0 dereceye dönmüş olmalıdır. (Servo kolları 0 dereceyi gösterecektir).
